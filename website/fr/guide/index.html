Prise en main

Tour d’horizon pratique : rack, transport, vue patch et projets.

Premier lancement

Le rack est une liste verticale de slots de plugins. Touchez + dans la barre d’application pour ajouter des instruments, des effets, des processeurs MIDI ou (sur bureau) des plugins VST3. Chaque slot peut être développé pour afficher son panneau ; la petite icône câble active la vue arrière du rack où les jacks MIDI et audio sont reliés.

Transport

La barre de transport règle le BPM, la signature rythmique, lecture/arrêt, tempo au tap et le métronome. Les patterns du générateur de batterie et les effets synchronisés au tempo suivent cette horloge. Ajustez le BPM avec le champ, les boutons pas à pas ou la molette lorsque c’est pris en charge.

Câbles patch

Dans la vue patch, chaque slot expose des jacks colorés : MIDI in/out, audio in/out, envois, retours, et (pour Jam Mode) sorties d’échelle ou d’accord. Appui long sur une sortie pour commencer un câble, relâchez sur une entrée compatible. Touchez un câble existant pour le retirer. L’application valide les types de ports et empêche les routages invalides (y compris les cycles audio).

Projets (.gf)

Utilisez le menu projet pour Ouvrir, Enregistrer ou Enregistrer sous. Les projets sont des fichiers JSON qui stockent l’ordre du rack, le graphe de câbles, le transport, les SoundFonts, les instantanés de paramètres VST3, les données du looper et les options par slot. Bureau et mobile enregistrent automatiquement la dernière session ; le build web n’a pas de stockage persistant — utilisez Enregistrer sous pour télécharger un .gf.

Préférences

Ouvrez les Préférences pour le comportement global du clavier (nombre de touches, gestes, CC d’aftertouch), la liste des périphériques MIDI, la cartographie CC (associer le matériel au volume, au balayage de patch, aux actions du looper, au cycle de gammes, etc.), les périphériques audio et les options visuelles de Jam Mode.
