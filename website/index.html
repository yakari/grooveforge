Soundfont · VST3 · Jam Mode

Forge your groove

GrooveForge is an open-source Flutter workstation: a plugin rack with multi-timbral SoundFont synthesis, descriptor-based audio and MIDI effects, hardware MIDI, an intelligent Jam Mode for scale practice, and a transport-synced drum pattern engine. Run it on Linux, macOS, Windows, Android — or try the browser build.
